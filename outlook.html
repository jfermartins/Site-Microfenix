Outlook

O curso Outlook apresenta as características e as ferramentas essenciais do Microsoft Outlook a fim de ensinar o melhor aproveitamento de suas funcionalidades. O aluno é introduzido aos elementos que compõem a interface do software e seus modos de exibição, além de recursos de gerenciamento de e-mails, contatos, calendários e tarefas.

Os Cursos de Softwares e Tecnologias Microsoft oferecidos pela Microfênix foram desenvolvidos de acordo com a exclusiva Metodologia de Ensino Microfênix, sendo ideais para o aprendizado prático e uso profissional.

Pré-requisito

  • Possuir conhecimentos básicos em Windows (salvar, copiar, colar, criar e renomear pastas)
  • Idade mínima: 14 anos
  • Escolaridade mínima: 6º ano do ensino fundamental
Carga horária: 08 horas

Informações do curso:

  • A interface do Outlook: Barra de ferramentas de acesso rápido, Barra de títulos, Ajuda do Microsoft Outlook, Barra de status, Faixa de opções (Guia Arquivo).
  • Modos de visualização.
  • Outlook Hoje.
  • Painel de Pastas: Modo de exibição de Pastas, Painel Favoritos.
  • Barra de tarefas pendentes.
  • Criando um contato: Catálogo de endereços, Grupo de contatos.
  • Visualizando os contatos.
  • Exportando contatos.
  • Importando contatos.
  • Imprimindo contatos.
  • Utilizando o diário: Registrando informações no diário.
  • Criando uma tarefa.
  • Gerenciando tarefas: Concluindo uma tarefa, Ativando uma tarefa concluída, Tarefas recorrentes, Excluindo tarefas, Aplicando um sinalizador para acompanhamento, Adicionando um lembrete.
  • Atribuindo uma tarefa: Respondendo uma tarefa atribuída.
  • Modos de exibição das tarefas: Barra de Navegação.
  • Classificação das tarefas.
  • Anotações.
  • Mensagens de e-mail: Enviando uma mensagem de e-mail (Criando uma assinatura, Solicitando uma notificação de entrega ou de leitura), Salvando uma mensagem, Mensagens com anexos (Salvando um anexo), Modos de visualização das mensagens (Modo Conversa), Classificando mensagens como não lidas, Pesquisando itens no Outlook, Respondendo uma mensagem de e-mail (Respostas automáticas), Encaminhando uma mensagem de e-mail, Anexando um arquivo ou item à mensagem, Imprimindo uma mensagem de e-mail, Apagando uma mensagem de e-mail, Filtro de lixo eletrônico, Recurso AutoArquivar.
  • Agendando um compromisso: Alterando um compromisso.
  • Agendando uma reunião: Respondendo uma mensagem com solicitação de reunião.
  • Modos de exibição do Calendário: Horário comercial.
  • Compartilhando informações de calendário.
  • Imprimindo um calendário.

Documentos necessários para matrícula:

Documento de Identidade (RG) (apresentação do original).

CPF ou documento que possua a numeração do CPF (apresentação do original).
